# Romer
-5 bar mechanism
-4 waveshare servo motors
-4 tap (2 for each player)
-20mm*20mm sigma metals
-plexiglass for coverage
-camera
-raspberry pi
-roller
Needs
-Game should be a closed box
-Ball should not stop in any moment, If it stops 5 bar mechanism should be able to move the ball
-There should be no external human intervention.
